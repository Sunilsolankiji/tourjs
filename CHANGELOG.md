# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-30

### Added

- Initial release of tourjs
- Core tour functionality with step-based navigation
- Customizable overlays and highlights
- Keyboard navigation support
- Smooth animations
- Multiple popup positioning options
- Progress tracking
- Callback hooks (onNextClick, onPreviousClick, onClose)
- Configurable buttons and text
- Responsive design for all screen sizes
- Apache 2.0 license

### Features

- Lightweight, zero-dependency library
- Pure vanilla JavaScript
- CSS customization support
- Backdrop click to close option
- Smooth scroll functionality
- Animation duration configuration
- Stage radius and padding options
- Custom CSS classes support

### Documentation

- Comprehensive README
- API documentation PDF
- Code of Conduct
- Contributing guidelines
- License file
